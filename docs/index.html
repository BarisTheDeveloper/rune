Rune

Modern Roblox development toolkit. Bidirectional sync between filesystem and Studio. No Rojo required.

Get Started View on GitHub

Features

Everything you need for Roblox development.

🔄

Real-time Sync

Filesystem changes reflect instantly in Studio via WebSocket or HTTP polling.

🎨

Minimal Plugin

VS Code-inspired dark UI with searchable file tree and activity log.

🔔

Toast Notifications

Connect, sync, and error alerts directly in Studio.

↩️

Undo System

Rollback up to 20 recent changes with one click.

📦

Zero Dependencies

Custom RBXMX builder — no Rojo, no external tools needed.

⏯️

Play Mode Safe

Auto-pauses HTTP polling during playtesting.

Install

Get up and running in seconds.

CLI

git clone https://github.com/BarisTheDeveloper/rune.git
cd rune
pnpm install
npx tsc

Studio Plugin

1
Run node install-plugin.mjs to build and install automatically
2
Or download RunePlugin.rbxmx from Releases
3
Place in %LOCALAPPDATA%\Roblox\Plugins\
4
Restart Roblox Studio — look for ◆ RUNE in Plugins tab

Bidirectional Sync

Changes go both ways.

DirectionTriggerResult
VS Code → StudioSave fileScript source updates instantly
Studio → VS CodeEdit script in StudioFile written back to disk

Commands

CLI at your fingertips.

CommandDescription
rune initCreate new project with full Roblox folder structure
rune watchWatch files + start sync server
rune syncStart sync server only
rune buildBuild .rbxlx place file
rune installInstall packages or plugins
